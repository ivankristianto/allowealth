# End User Onboarding

## 1. Complete your profile

Set your name, locale, and default currency preferences.

## 2. Connect your first account

Add at least one account so tracking and reports can start.

## 3. Add your first transactions

Capture a few recent transactions to establish baseline activity.

Continue with [End User Daily Workflow](/end-users/daily-workflow/).
