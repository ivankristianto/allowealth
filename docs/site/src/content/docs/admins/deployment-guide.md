## Pre-deployment

- Validate configuration and permissions.
- Confirm monitoring and alerts are enabled.

## Rollout

- Deploy in a controlled window.
- Run smoke checks across key workflows.

## Post-deployment

- Review health metrics.
- Record release notes and owner follow-ups.
