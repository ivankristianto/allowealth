Dashboard displays your complete financial picture on one screen. Check this page daily to monitor financial health.

![Dashboard Overview](https://images.allowealth.io/dashboard.png)

## What Dashboard Shows

### Spending Summary Card

![Spending Summary Card](https://images.allowealth.io/dashboard-spending-summary.png)

The top card displays current month performance:

- **Spent** - Total expenses this month
- **Budget** - Total budget allocation
- **Income** - Total income received
- **Remaining** - Budget left for the month

A progress bar shows spending versus budget. Green indicates healthy spending. Yellow approaches limits. Red indicates exceeded budgets.

### Spending Analysis Chart

![Spending Analysis Chart](https://images.allowealth.io/dashboard-spending-chart.png)

A donut chart breaks down spending by category. The top four categories display individually. Remaining categories group as "Other."

Hover over chart segments to see exact amounts and percentages.

### Recent Transactions

The last ten transactions appear in a list. Each row shows:

- Transaction date
- Category icon and name
- Description
- Account
- Amount (expenses in red, income in green)

Click any transaction to edit details. Click **View All** to see the complete transaction history.

### Account Summary

![Account Summary](https://images.allowealth.io/dashboard-account-summary.png)

Total balances across all accounts display by currency. The card shows:

- **Assets** - Sum of liquid and non-liquid accounts
- **Debt** - Sum of credit cards and loans
- **Net Worth** - Assets minus debt

Click **View Details** to navigate to the full Accounts page.

### Cash Flow

Recent income and expense activity appears chronologically. This section highlights:

- Recent deposits and withdrawals
- Upcoming recurring bills
- Largest transactions this period

### Recurring Bills Summary

![Recurring Bills Summary](https://images.allowealth.io/dashboard-recurring-bills.png)

Upcoming recurring transactions display with their due dates. This helps you anticipate upcoming expenses.

## Daily Dashboard Routine

Spend five minutes each morning reviewing these items:

1. **Check the spending summary** - Confirm the month-to-date total aligns with your expectations
2. **Review recent transactions** - Spot any duplicates or miscategorized items
3. **Scan account balances** - Verify no unexpected changes occurred overnight
4. **Note upcoming bills** - Prepare for scheduled recurring transactions

## Interacting with Dashboard

### Switch Currency

If your workspace uses multiple currencies, click the currency selector to view data in your preferred currency.

### Refresh Data

Dashboard data updates automatically when you add or edit transactions. Refresh the browser if data appears stale.

### Navigate to Details

Click any summary card to access detailed views:

- Spending summary → Budget page
- Recent transactions → Transactions page
- Account summary → Accounts page
- Recurring bills → Recurring page

## Troubleshooting

### Spending Seems Too High

1. Click the spending amount to view transactions
2. Check for duplicate entries
3. Verify transaction dates fall in the current month
4. Confirm all transactions use the correct category

### Account Balance Looks Wrong

1. Click **View Details** in the accounts section
2. Compare with your external bank statement
3. Check for missing transactions
4. Verify the account currency setting

### Budget Status Seems Off

1. Click the budget amount to navigate to the Budget page
2. Review category-level allocations
3. Confirm all transactions categorize correctly
4. Adjust budget limits if consistently exceeded

### Data Loads Slowly

Dashboard aggregates data from multiple sources. If loading takes more than a few seconds:

1. Check your internet connection
2. Try refreshing the page
3. Contact support if slowness persists

## Customization

Dashboard automatically adapts to your workspace:

- Multi-currency workspaces show currency selectors
- Workspaces with recurring transactions show the bills widget
- Member information appears in multi-user workspaces

## Security Note

Dashboard displays sensitive financial data. Ensure you:

- Log out when finished on shared computers
- Enable MFA in Security settings
- Do not share screenshots containing account numbers
