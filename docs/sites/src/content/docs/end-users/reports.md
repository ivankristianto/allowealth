Reports help you understand patterns, not just totals.

![Reports screenshot placeholder](/images/user-guides/reports-placeholder.svg)

## What you can do here

- Switch between **monthly** and **yearly** analysis.
- Compare income vs expenses over time.
- Inspect category-level spending and budget behavior.
- Review spending by member.

## Weekly report review

1. Open **Reports**.
2. Start in **monthly** view for current behavior.
3. Check top categories and trend charts.
4. Open **Budget** if any category is drifting.
5. Switch to **yearly** view for bigger trajectory checks.

:::tip[💡 Tip]
Use reports after transaction cleanup. Report quality is only as good as your transaction data.
:::

## Drill-down workflow

- Find a suspicious category in the category table.
- Open drill-down details.
- Confirm which transactions created the spike.
- Return to **Transactions** to correct source records if needed.
