Transactions are the source of truth for budgets, reports, and forecasts.

![Transactions screenshot placeholder](/images/user-guides/transactions-placeholder.svg)

## Key actions

- Add new **expense** or **income** entries.
- Filter by type, category, account, member, and month.
- Import transactions from CSV.
- Export your filtered results to CSV.
- Use bulk selection for cleanup tasks.

## Add a transaction in seconds

1. Open **Transactions**.
2. Click **Expense** or **Income** in the action bar.
3. Fill in amount, date, category, and account.
4. Save and verify it appears in the list.

## Keep data clean

- Review uncategorized or miscategorized transactions weekly.
- Use search plus filters together to isolate problem records.
- Export before large cleanup sessions if you want a backup.

:::caution[⚠️ Caution]
Import files should match the expected template format. Invalid columns can fail import or produce incorrect mappings.
:::
