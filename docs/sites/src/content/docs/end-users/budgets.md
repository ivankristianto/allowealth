Use Budgets to turn spending into a plan.

![Budgets screenshot placeholder](/images/user-guides/budgets-placeholder.svg)

## What you can do here

- Set or update budgets by category.
- Move through months with period controls.
- Import and export budget data.
- Copy current month budgets into the next month.
- Manage budget categories.

## Recommended monthly flow

1. Open **Budget** at the start of the month.
2. Create or adjust limits for key expense categories.
3. Mid-month, review warning or exceeded categories.
4. Before month-end, copy budgets forward if next month is ready.

:::tip[💡 Tip]
Use **Categories** in the budget action bar to keep category names clean and consistent. Better category hygiene means clearer reports.
:::

## Troubleshooting

- No budget rows shown: initialize budgets for uninitialized categories.
- Too many alerts: check duplicate transactions in **Transactions** first.
- Copy action disabled: the target month may already have existing budgets.
