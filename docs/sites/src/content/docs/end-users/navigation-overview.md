Use this page to quickly understand where to go in the app.

![Navigation screenshot placeholder](/images/user-guides/navigation-overview-placeholder.svg)

## Main navigation menu

- **Dashboard**: Your financial snapshot and onboarding progress.
- **Transactions**: Add, import, filter, edit, and export money activity.
- **Budget**: Plan limits by category and track budget status.
- **Accounts**: Manage wallets, bank accounts, investments, and debt accounts.
- **Reports**: Analyze trends, categories, and member spending.
- **Forecast**: Project future wealth based on contributions and return assumptions.
- **Calculators**: Run planning tools, including compound growth.
- **Settings**: Manage workspace preferences, members, notifications, and data actions.

## User menu (bottom-left profile area)

- **Manage account**: Update your profile and email.
- **Security**: Manage MFA, connected accounts, passkeys, and API keys.

:::tip[💡 Tip]
If you are unsure where to start, go to **Dashboard** first, then follow the guides below in sidebar order.
:::
