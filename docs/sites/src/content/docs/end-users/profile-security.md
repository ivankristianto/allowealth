## Profile tasks

- Update your display name and contact details.
- Change your email and complete verification if required.
- Review connected OAuth account state.

## Security tasks

- Enable and maintain MFA.
- Regenerate and store backup codes safely.
- Review connected accounts and passkeys.
- Create and revoke API keys when needed.

## Security baseline checklist

1. Enable MFA.
2. Save backup codes in a secure location.
3. Remove connected accounts you no longer use.
4. Rotate API keys you no longer trust.

:::tip[💡 Tip]
Treat API keys like passwords. Store them in a secret manager and never share them in chat or source control.
:::
