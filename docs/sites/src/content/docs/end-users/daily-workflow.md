These routines keep your data accurate without overwhelming your schedule.

## Daily Routine (5 Minutes)

Complete these tasks each day to maintain current data.

### 1. Add New Transactions

Record expenses and income as they occur.

1. Open **Transactions**
2. Click **Add Expense** or **Add Income**
3. Fill in amount, category, account, and description
4. Save

**Tip:** Use the mobile view for quick entry while shopping.

### 2. Review Uncategorized Items

Check for transactions missing categories.

1. In **Transactions**, filter by category "Uncategorized"
2. Select a transaction
3. Assign the correct category
4. Save changes

### 3. Check Dashboard for Anomalies

Scan the Dashboard for unexpected totals.

- Verify spending aligns with your memory
- Confirm account balances look correct
- Note any duplicate transactions

## Weekly Routine (15-20 Minutes)

Complete these tasks once per week, ideally on the same day.

### 1. Review Budget Status

Check which categories are approaching limits.

1. Open **Budget**
2. Review categories with warning indicators (yellow) or exceeded limits (red)
3. Adjust spending for the remainder of the week if needed
4. Consider increasing limits for categories that consistently exceed budgets

### 2. Reconcile Account Balances

Match Allowealth balances with external statements.

1. Open **Accounts**
2. Compare each account balance with your bank or credit card statement
3. Investigate discrepancies by checking for:
   - Missing transactions
   - Duplicate entries
   - Incorrect amounts
4. Add or correct transactions as needed

### 3. Categorize Bulk Transactions

Clean up any accumulated uncategorized items.

1. In **Transactions**, use filters to find uncategorized items
2. Use bulk selection for similar transactions
3. Apply the correct category to all selected items

## Monthly Routine (30-45 Minutes)

Complete these tasks at month-end or month-start.

### 1. Close the Previous Month

Finalize the month's financial record.

1. Open **Reports**
2. Switch to the previous month view
3. Review income versus expenses
4. Note any significant variances from budget
5. Export a CSV backup if desired

### 2. Plan Next Month's Budget

Adjust budgets based on upcoming expenses.

1. Open **Budget**
2. Navigate to the new month
3. Copy last month's budgets (click **Copy to Next Month**)
4. Adjust categories for known changes (holidays, insurance payments, etc.)

### 3. Review Trends

Analyze longer-term patterns.

1. Open **Reports**
2. Switch to **Yearly** view
3. Compare current month to previous months
4. Identify spending trends that need attention

### 4. Archive Old Accounts or Categories

Clean up unused items.

1. Review accounts you no longer use
2. Close inactive accounts (this preserves historical data)
3. Archive categories you no longer need

## Multi-User Workflows

When multiple people share a workspace:

### Daily

- Each user adds their own transactions
- Tag transactions with descriptions for clarity

### Weekly

- Designate one person to review and categorize shared expenses
- Hold a brief sync meeting to discuss budget status

### Monthly

- Review reports together
- Adjust shared budget categories
- Plan for upcoming shared expenses

## Common Issues and Solutions

### Totals Look Incorrect

**Symptoms:** Dashboard shows wrong balance or spending amount

**Solutions:**

1. Check for duplicate transactions (search by amount)
2. Verify transaction dates fall in the correct month
3. Confirm account currency matches the transaction currency
4. Check that transfers appear in both accounts (if applicable)

### Too Many Uncategorized Transactions

**Symptoms:** Large percentage of transactions lack categories

**Solutions:**

1. Create clearer category names
2. Set up recurring transactions for regular merchants
3. Import transactions with category mappings from your bank

### Budget Alerts Feel Wrong

**Symptoms:** Categories show exceeded budgets but spending seems normal

**Solutions:**

1. Check for duplicate transactions inflating totals
2. Verify the budget amount is realistic
3. Confirm transactions use the correct category
4. Adjust budget limits for categories that consistently trigger alerts

## Time-Saving Tips

- **Use recurring transactions** for monthly bills and subscriptions
- **Import bank data** weekly instead of manual entry
- **Set up category rules** by consistently using the same descriptions
- **Use bulk edit** for transactions from the same merchant
- **Export data** monthly for external backup
