This routine keeps your data accurate without heavy overhead.

## Daily (5-10 minutes)

1. Review new uncategorized transactions.
2. Fix category mismatches and duplicates.
3. Check dashboard totals for obvious outliers.
4. Resolve pending budget alerts.

## Weekly (20-30 minutes)

1. Review top category spend deltas.
2. Adjust category budgets for the next cycle.
3. Reconcile account balances against external statements.
4. Export CSV if you need a backup snapshot.

## Monthly (30-45 minutes)

1. Close prior month and record key notes.
2. Review trend charts and report deltas.
3. Archive stale categories/accounts where needed.

:::note[📝 Note]
If your workspace has multiple contributors, run your weekly review before your team sync so everyone works from the same baseline.
:::

## Common issues

### Totals look wrong

- Check for duplicate imports.
- Confirm timezone and month filters.
- Reconcile account transfer entries.

### Too many uncategorized transactions

- Create clearer category naming standards.
- Batch-edit recurring merchants weekly.
