Use this reference to find the right tool for any financial task.

## Main Menu

The sidebar navigation contains all primary features:

### Dashboard

Your daily financial snapshot. Review spending, account balances, and recent activity. Check this screen each morning for a quick financial health check.

**Best for:** Daily check-ins, spotting unusual activity, quick transaction reviews

### Transactions

The center of all money activity. Add income and expenses, import bank data, export records, and clean up miscategorized entries.

**Best for:** Recording transactions, bulk imports, data cleanup

### Budget

Set monthly spending limits by category. Track progress through the month and receive alerts when categories exceed thresholds.

**Best for:** Monthly planning, spending control, category-level limits

### Accounts

Manage your financial accounts grouped by type: liquid (cash and checking), non-liquid (investments), and debt (credit cards and loans). Transfer funds between accounts and view historical snapshots.

**Best for:** Balance tracking, account transfers, net worth monitoring

### Reports

Analyze financial patterns with monthly and yearly views. View income versus expenses, category breakdowns, and member spending summaries.

**Best for:** Monthly reviews, trend analysis, spending investigations

### Forecast

Project future wealth based on current savings rates and return assumptions. Visualize how today's decisions affect tomorrow's outcomes.

**Best for:** Long-term planning, retirement preparation, goal setting

### Calculators

Run financial scenarios with built-in tools. Calculate compound interest growth and compare investment options.

**Best for:** Quick calculations, scenario planning, education

### Recurring

Manage repeating transactions like subscriptions, rent, and salary. View pending occurrences and confirm or skip upcoming items.

**Best for:** Subscription tracking, bill management, automation

### Settings

Configure workspace preferences, invite team members, and manage notification settings. Access data maintenance tools.

**Best for:** Workspace setup, member management, preference changes

## User Menu

Access the user menu from your profile picture at the bottom of the sidebar:

### Profile

Update your name, email address, and contact information. Manage pending email changes.

**Best for:** Personal information updates, email verification

### Security

Enable multi-factor authentication, manage passkeys, review connected accounts, and generate API keys for external integrations.

**Best for:** Account protection, access management, API access

## Navigation Tips

- Use keyboard shortcuts where available
- The sidebar collapses on mobile devices
- Recent transactions appear on Dashboard for quick reference
- Budget alerts link directly to relevant categories

## Decision Guide

| Task                             | Go To                      |
| -------------------------------- | -------------------------- |
| Record a purchase                | Transactions → Add Expense |
| Check spending this month        | Dashboard or Reports       |
| See if I am over budget          | Budget                     |
| Update account balance           | Accounts                   |
| Add a new credit card            | Accounts → Add Account     |
| See spending trends              | Reports                    |
| Calculate investment growth      | Calculators                |
| Pay a recurring bill             | Recurring                  |
| Invite a family member           | Settings → Members         |
| Enable two-factor authentication | Security                   |

Start with [Dashboard](./dashboard/) for your daily routine, then explore other sections as needed.
